HTML

Function of HTML in a webpage

HTML (HyperText Markup Language) provides structure to the webpage.

How HTML is written

A webpage is made up of many different HTML elements.

  • The Opening tag consists of the element's name and attributes surrounded by angle brackets.
  • The Closing tag consists of a forward slash and then the element name.
  • In between the opening tag and the closing tag contains the content.
  • Elements may also have attributes, which contain extra information about the element

Every HTML page consists of the following:

  • <!DOCTYPE html>: tells the browser that document is an HTML5 document
  • <html>: encloses all the content on the page.
  • <head>: contains information about the page that is not the content as search engine keywords, character set of the page, link to stylesheets, and more.
  • <meta charset="UTF-8">: sets the document's character set to UTF-8.
  • <title>: sets the page title of the webpage, which appears in the browser's tab.
  • <body>: wraps the content that is displayed on the webpage.

Example of an HTML document:

                
    <!DOCTYPE html> 
    <html lang="en"> 
    <head> 
        <meta charset="UTF-8"> 
        <title>Document</title> 
    </head>
    <body> 

        <p>This is a paragraph.</p>

    </body> 
    </html>

CSS

Function of CSS in a webpage

CSS (Cascading Style Sheets) is used to style a webpage. It also controls how elements should be displayed on the webpage.

How CSS is written

In CSS, you define rules for selectors and associate them with properties to style them.

Here's how the CSS syntax looks like:

                
    selector {

        property: value;

    }
  • selector {}: selects the html element that you want to change the style of.
  • property: : the name of the property that you want to change.
  • value: controls how much of the property you want to change.

Example of a CSS Rule:

                
    p {

        font-size: 20px;

    }

The paragraph element is selected and the font size of it is increased to 20px.
