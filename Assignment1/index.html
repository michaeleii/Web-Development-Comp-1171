HTML

Function of HTML in a webpage

HTML provides structure in a webpage.

How HTML is written


    <!DOCTYPE html> 
    <html lang="en"> 
    <head> 
        <meta charset="UTF-8"> 
        <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
        <title>Document</title> 
    </head>
    <body> 

     Content goes here...

    </body> 
    </html>

CSS

Function of CSS in a webpage

CSS is used to style a webpage. It also controls how elements should be displayed on the webpage.

How CSS is written


    selector {

        property: value;

    }
